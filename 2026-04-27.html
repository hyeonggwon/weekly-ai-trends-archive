Google이 Anthropic에 최대 $40B 투자 — Amazon $5B와 같은 주에 발표, 모두 $350B 밸류에이션

Google은 4/24 Anthropic에 즉시 $10B 현금 투자를 확정했고 성과 목표 달성 시 추가 $30B를 약속함. 같은 주 4/20에는 Amazon이 $5B 추가 투자와 $100B AWS 약정을 공개. 두 거래 모두 $350B 밸류에이션 동일. Claude Code 사용자 입장에서 보는 함의: 향후 1~2년 컴퓨트가 부족해서 Claude Code가 느려지거나 한도가 더 빡빡해질 가능성은 줄어든 셈. 단 가격 정책은 별개라 모니터링 필요.

Claude Code 2.1.117(4/22)부터 Opus 4.7이 디폴트, 1M 컨텍스트 정상 인식 패치

Pro/Max 사용자의 Claude Code 디폴트가 Opus 4.7로 전환되고, /context 퍼센트가 200K 기준으로 잘못 계산돼 너무 일찍 auto-compact 되던 버그가 수정됨. Pro/Max 디폴트 effort가 high로 상향. 즉, 이번 주부터 형권님이 Claude Code 새 세션을 열면 별도 설정 없이 Opus 4.7 + 1M 컨텍스트 + high effort가 기본값. 긴 코드베이스 분석이 한층 자연스러워지고, 매번 /model로 옮겨다닐 필요가 사라짐.

GitHub Copilot Pro 신규 가입 일시 중단·Opus 제외 — 에이전트 부하가 가격 모델을 깨다

4/20 GitHub은 Copilot Pro·Pro+·Student 신규 가입을 일시 중단하고, Opus 4.7을 Pro에서 제거(Pro+에서만 유지)함. 사유는 명확함: 에이전트 워크플로우의 컴퓨트 소비가 기존 정액 구조를 무너뜨림. 한 주 전후로 Cursor도 Pro 가격을 $20에 동결하면서 상위 티어(Pro+ $60, Ultra $200)로 부하를 분리. Claude Code Pro/Max는 여전히 정액이지만 5시간 캐시·prompt 캐시 단축 같은 조정이 이어질 가능성에 대비할 것.
