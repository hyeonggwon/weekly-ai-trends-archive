Claude Code가 Pro($20)에서 잠깐 사라졌다 되돌아왔다

Anthropic이 4/21 조용히 pricing 페이지에서 Claude Code를 Max 전용(최저 $100/mo)으로 바꿨다가 몇 시간 뒤 되돌렸음. Head of Growth는 '신규 가입 2%만 대상인 테스트'라고 해명했지만 공식 공지 없이 진행돼 신뢰 이슈가 붉어짐. Simon Willison은 OpenAI Codex가 무료·$20 플랜을 유지하는 점을 들어 대안 카드로 부상했다고 지적. Claude Code 사용자에겐 당장의 요금 변화는 없지만, 개인·교육 사용자 층의 가격 리스크가 공식화된 한 주.

Claude, 15개 소비자 앱 커넥터로 '일상의 관문' 선언

4/24 Claude가 Spotify·Uber·Uber Eats·Instacart·TurboTax·Booking.com·StubHub·Audible·Resy·AllTrails 등 15개 개인용 커넥터 출시, 전 플랜·모바일 베타 동시 지원. 개인 커넥터 데이터는 학습에 쓰지 않고 결제·예약 전에 반드시 사용자 확인을 받는 구조. Anthropic은 같은 주 'Claude는 광고 없이 유지한다'는 선언문도 냈음. Claude Code 사용자에겐 직접 영향은 작지만, 에이전트가 '브라우저를 대체하는 layer'로 가는 방향이 제품에서 한번 더 확정된 시점.

MCP STDIO 설계 결함으로 20만 서버 RCE 위험 — Anthropic은 'by design'

OX Security가 4/15 Anthropic 공식 MCP SDK(Python/TS/Java/Rust) 전반에 걸친 '설계상' RCE 취약점을 공개. 영향 범위 150M+ 다운로드·7,000+ 공개 서버·최대 20만 인스턴스. MCP 레지스트리 11곳 중 9곳이 악성 서버로 '포이즈닝' 가능했으며 Windsurf·Cursor·LiteLLM·LangFlow 등도 RCE 영향. Anthropic은 STDIO 실행 모델이 의도된 동작이며 sanitization은 개발자 책임이라고 답변. Claude Code 사용자에겐 플러그인·MCP 서버 설치 전 공급망 검증이 '선택'에서 '기본기'로 바뀐 주.
